# DreamSpace Kids (iOS Xcode)

A SwiftUI iOS app for DreamSpace that teaches kids core coding ideas with levels, mini challenges, onboarding, and progress badges.

## Open in Xcode
1. Open `DreamSpaceKids.xcodeproj` in Xcode.
2. Select the **DreamSpaceKids** scheme.
3. Pick an iPhone or iPad simulator.
4. Run (▶).

> If you see a signing warning, set your Team in **Signing & Capabilities**.

## Project layout
- `DreamSpaceKids/App`: SwiftUI app entry point and root view
- `DreamSpaceKids/Models`: lesson and challenge data
- `DreamSpaceKids/Stores`: progress tracking store
- `DreamSpaceKids/Views`: UI screens
- `DreamSpaceKids/Assets.xcassets`: app icons

## What’s new
- Onboarding flow with kid-friendly intro screens
- Level progression where each level unlocks a new skill
