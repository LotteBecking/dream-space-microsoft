💡 What is a Conditional?

A conditional is a rule that makes a decision. The computer checks if something is true, then decides what to do!

IF condition is true → THEN do this ELSE do that instead

🌧️ IF raining THEN umbrella ELSE sunglasses
🔋 IF battery low THEN charge ELSE keep going
